# BookMyShow-Concurrency-Simulation
A terminal-based Java application simulating a Multiplex Ticket Booking System. Demonstrates core Java multithreading, thread synchronization, and resolving race conditions during concurrent bookings.
# 🎬 Multiplex Ticket Booking System (Concurrency Simulation)

A core Java application that simulates a real-time movie ticket booking system (like BookMyShow). This project was built to practically demonstrate and solve the **"Race Condition"** that occurs when multiple users try to book the same seat simultaneously.

## 🚀 Key Features
* **Live Seat Tracking:** View the current number of available seats out of 50
